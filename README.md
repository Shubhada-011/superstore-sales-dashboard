# Super Store Sales Dashboard

## 📌 Project Overview
Business Analyst portfolio project analyzing ~6K sales records using Power BI.  
Data was cleaned and transformed using **Power Query**, then visualized in interactive dashboards to highlight regional sales, customer segments, payment modes, and shipping performance. A short-term sales forecast was also included.

## 📊 Key Insights
- West region contributes 33% of sales, South only 16% → Target promotions in South to boost performance.
- COD accounts for 43% of payments → Encourage digital payments via discounts to reduce cash-handling risk.
- Standard Class shipping yields highest profit → Expand Standard Class options and optimize logistics for cost efficiency.

## 🎯 Deliverables
- Data cleaning and transformation (Power Query)
- Interactive dashboard (Power BI)
- 15-day sales forecast
- Actionable insights & recommendations for business growth

## 🛠️ Tools Used
- Power BI (with Power Query)
